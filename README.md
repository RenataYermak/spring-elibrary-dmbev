## **Task**

### **Library**

The reader has the opportunity to develop the search and order Books in the Catalog. The librarian gives the Reader a
Book for a Season Ticket or in the Reading Room. The book can be distributed in the Library in one or multiple copies.
Administrator manages Readers and Books.
***
### *User:*
- log in/log out
- profile data edit
- switch language
- view book catalog list
- search catalog books
- order book for reading room or season ticket
- view all own orders list
- return orders
- delete orders
***
### *Admin:*
- __all user's allowed services__
- registration user
- edit users
- delete users
- view all users list
- add a new book
- edit the book
- send email after registration users
***
### Components used for the project:
- Java 17
- Maven
- Spring Boot(Core, Data, Web, Secutrity, AOP)
- Hibernate
- Querydsl
- HTML, CSS
- Docker
- Database: PostgreSQL
- Logger: Logback
- Tests: JUnit 5, Mockito
